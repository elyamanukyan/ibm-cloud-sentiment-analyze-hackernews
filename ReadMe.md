# Python Assignment

**OS:** Any OS

**Used Technologies:**
- Python 3.x
- IBM Cloud Sentiment Analyze API

## Installation
Clone repository
```
git clone git@github.com:elyamanukyan/ibm-cloud-sentiment-analyze-hackernews.git
```

In project root dir, install dependencies:
```
pip install -r requirements.txt
```

Usage:
Run `main.py` file.
